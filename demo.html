pull-quote Web Component Demo

Source code available at https://github.com/miklb/pull-quote/

Pull quotes can be useful in calling out a specific quote from a larger body of text. This web component allows you to easily add a pull quote to your site. The component is built using web components, so it can be used in any web project, regardless of the framework or CMS you are using. It's pretty straightforward and doesn't require any complex setup or configuration to get started.
